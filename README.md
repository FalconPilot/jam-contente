# JAM-Contente Game Engine

Do you like working on games? Do you need something quick for game jams? Do you want to quickly bootstrap a project with no frills?

Then the JAM-Contente engine... well, it might be of interest to you. Named in reference to the [Jamais Contente](https://en.wikipedia.org/wiki/La_Jamais_Contente), this engine is built with the simple goal in mind of letting people get started with a basic game engine using React for games that wanna rely on browser-based technologies.

- React front-end, built with Vite
- Tons of utility functions to handle asset loading
- State management abstractions to provide plug'n play persistent data handling

## Peer dependencies

- React
- Vite

## Recommended dependencies

- TypeScript
- A library to handle decoding (such as [Zod](https://www.npmjs.com/package/zod))
